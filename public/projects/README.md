Project Screenshots

Place your custom project screenshots in this folder.

Expected filenames used by `components/projects.tsx`:
- heart-sync.png
- socket-chat-app.png
- question-builder.png
- game-nest.png
- cooller.png
- weather-summary.png
- speed-booster-extension.png
- fake-news-detector.png
- apna-market.png

Recommended image specs:
- Format: PNG or JPG
- Aspect ratio: 16:9 or close
- Size: 1280x720 (or higher)
- Keep file size under ~500 KB for faster loading

Fallback behavior in UI:
1. If `screenshot` exists for a project, it is shown.
2. Otherwise, GitHub preview image is used (for GitHub links).
3. Otherwise, a gradient placeholder card is shown.
